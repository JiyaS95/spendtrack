# SpendTrack
SpendTrack is a simple command-line expense tracker written in java.
It helps you record, view, delete, and analyze your spending while keeping your data saved between sessions.

## Features
- Add, view, and delete expenses
- Expenses automatically saved as **JSON**
- Optional export of expenses to **CSV**
- View spending summaries by category, month, or year
- Search expenses by keyword or date
- View summaries with **ASCII bar charts**

## Setup & Running
### Compile
- javac src/main/java/Expense.java src/main/java/SpendTrack.java

### Run
- java -cp src/main/java SpendTrack
